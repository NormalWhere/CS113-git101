- ชนิดตัวแปร 
    ต้องประกาศตัวแปรพร้อมกำหนดชนิดข้อมูลให้กับตัวแปรด้วย
    int  = จำนวนเต็ม
    float  = จำนวนจริง
    char = ตัวอักขระ
    double = สามารถ เก็บจำนวนจริงที่มีจำนวนมากๆได้
    long = เพิ่มขนาดการเด็บข้อมูล
    unsigned = เป็นการเพิ่มการเก็บข้อมูลค่าบวก
- สามารถตรวจสอบขนาดของชนิดแต่ละตัวได้ด้วยคำสั่ง sizeof(ตัวแปร)
- ค่าคงที่ข้องเลขฐาน 2 จะขึ้นต้นด้วย 0b
- ค่าคงที่ข้องเลขฐาน 8 จะขึ้นต้นด้วย 0 (%o)
- ค่าคงที่ข้องเลขฐาน 16 จะขึ้นต้นด้วย 0x (%x / %X)
- getchar() รับข้อมูลเป็น สตริง จาก standard input
- putchar() ส่งอักขระที่อยู่ในตัวแปรใดๆไปยัง standard output


*** % ไม่สามารถใช้กับ float หรือ double ได้ ***
*** ผลหารจะเป็นจำนวนเต็มถ้าตัวหารและตัวถูกหารเป็นจำนวนเต็ม ***

-ตัวดำเนินการ
    relational operators > >= < <= == !=
    logical operators && || !
