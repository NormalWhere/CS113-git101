array คือ กลุ่มตัวแปรที่ถูกเก็บอยู่รวมกันโดยตัวแปรต้องเป็นประเภทเดียวกัน
โดยเมื่อเรียกข้อมูล ข้อมูลแต่ละตัวจะมี index ประจำข้อมูล บอกตำแหน่งที่อยู่ใน array
การประกาศ int A[10] ; จะมี index ตั่งแต่ 0 - 9
ขนาดของ array ควรเป็น ค่า constant คือจะไม่สามารถเลี่ยนแปลงตลอดการทำงานของโปรแกรม
#define arraysize 5 คือ เปลี่ยนอักษรทั้งหมดที่เหมือนกับที่กำหนดให้มีค่าเท่ากับ 5
*** การส่งค่าตัวแปรใน array เป็นการส่งแบบ pass by reference ***
การ pass by reference เป็นการส่งตำแหน่งของข้อมูลจากผู้เรียกไปให้ฟังชั่น เมื่อเปลี่ยนค่าตัวแปรนั้นในฟังก์ชั่น ข้อมูลต้นตำหรับจะถูกเปลี่ยนด้วย
ตัวแปรหนึ่งตัวกินเนื้อที่ใน memory ตามชนิดของข้อมุล

array 2D เป็นการสร้างตารางแบบสองชั้น
A[i][j] โดย i จะเป็นตัวกำหนดจำนวนของแถวแนวนอน (row)
            j จะเป็นตัวกำหนดจำนวนของแถวแนวตั้ง (column)
